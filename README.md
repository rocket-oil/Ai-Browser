# Headless Browser / 无头浏览器

## 功能 / Features

- **截图 (全页或指定元素)**  
  Screenshot (full page or specific element)

- **PDF 导出**  
  PDF Export

- **抓取页面 HTML**  
  Fetch Page HTML

- **捕获网络请求 (HAR 格式)**  
  Capture Network Requests (HAR format)

- **执行自定义 JS 表达式**  
  Execute Custom JS Expressions

- **同步返回浏览器控制台日志**  
  Return Browser Console Logs

# 安装依赖 / Install dependencies
